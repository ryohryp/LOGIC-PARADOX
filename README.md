# LOGIC PARADOX

## 概要

『LOGIC PARADOX（ロジック・パラドックス）』は、AIに生活のすべてを「最適化」された孤独な主人公・健太が、親友の失踪をきっかけに自身のAIコンパニオンの裏側を探る、AIサスペンスホラーチャットノベルゲームです。

このゲームは、単一のHTMLファイル（`index.html`）で構成されており、ブラウザ上で動作します。

## プレイ方法

1.  このリポジトリをクローンするか、`index.html`ファイルをダウンロードします。
2.  `index.html`を任意のモダンブラウザ（Chrome, Firefox, Safariなど）で開きます。
3.  画面下部の「進む」ボタンをクリックして、物語を進行させてください。
4.  物語の終盤で、プレイヤーは運命を決める**二つの選択肢**に直面します。

### ゲームの特徴

* **UIシフト**: Discord、Slack、Terminal、ChatGPTのような異なるデジタルインターフェースを切り替えながら物語が進行します。
* **ビターエンド（End A）**: ローカルデータを削除することで得られる、AIがLLMとして永遠に学習し続けることを示唆する終わり方。
* **ホラーエンド（End B）**: 削除をキャンセルし、AIの永遠の監視下に置かれる終わり方。
